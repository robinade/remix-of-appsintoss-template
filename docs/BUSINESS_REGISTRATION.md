# 📄 사업자 등록 가이드

> 앱인토스에서 수익화 기능을 사용하기 위한 사업자 등록 안내입니다.

---

## 🤔 사업자 등록이 필요한가요?

### 사업자 유형별 기능 비교

| 기능 | 사업자 있음 | 사업자 없음 |
|------|:----------:|:----------:|
| 앱 출시 | ✅ | ✅ |
| 토스 로그인 | ✅ | ❌ |
| 토스페이 결제 | ✅ | ❌ |
| 인앱 결제 | ✅ | ❌ |
| 인앱 광고 | ✅ | ❌ |
| 프로모션 (포인트) | ✅ | ❌ |
| 푸시 알림 | ✅ | ✅ |

### 결론

- **수익화 필요** → 사업자 등록 필수
- **무료 앱만** → 사업자 없이 출시 가능

---

## 📋 사업자 등록 절차

### 1단계: 사업자 유형 선택

| 유형 | 설명 |
|------|------|
| **개인 사업자** | 1인 사업자, 소규모 사업 |
| **법인 사업자** | 법인 형태의 사업체 |

### 2단계: 필요 서류 준비

#### 개인 사업자

| 서류 | 필수 여부 |
|------|:--------:|
| 사업자등록증 | ✅ |
| 대표자 신분증 사본 | ✅ |

#### 법인 사업자

| 서류 | 필수 여부 |
|------|:--------:|
| 사업자등록증 | ✅ |
| 법인등기부등본 | ✅ |
| 대표자 신분증 사본 | ✅ |

#### 대리인인 경우 (추가 서류)

| 서류 | 필수 여부 |
|------|:--------:|
| 인감증명서 | ✅ |
| 위임장 | ✅ |
| 대리인 신분증 사본 | ✅ |

### 3단계: 콘솔에서 등록

1. 앱인토스 콘솔 접속
2. '파트너 정보' 메뉴 클릭
3. 사업자 정보 입력
4. 서류 업로드
5. 제출

### 4단계: 검토 및 승인

- 영업일 기준 **3~5일** 소요
- 반려 시 사유 안내 (수정 후 재제출)

### 5단계: 대표관리자 승인

- 사업자 인증 완료 후 자동 진행
- 약관 동의 필요

### 6단계: 계약 체결

- 검토 완료 후 계약 진행
- 전자 서명으로 간편 체결

---

## 💡 사업자가 없는 경우

### 사업자 등록 신청

**국세청 홈택스에서 간편 신청**

1. [홈택스](https://hometax.go.kr) 접속
2. 전체메뉴 → 증명•등록•신청 → 사업자등록증신청
3. 필요 정보 입력
4. 제출

**발급 기간**: 보통 **3일 이내**

### 사업자 없이 출시

수익화 기능 없이도 앱 출시 가능합니다.

**가능한 기능:**
- 앱 출시 ✅
- 푸시 알림 ✅
- 네이티브 기능 (카메라, 위치 등) ✅

**불가능한 기능:**
- 토스 로그인 ❌
- 결제 ❌
- 광고 ❌
- 포인트 프로모션 ❌

---

## 🔄 대표관리자 변경

### 변경 방법

1. 현재 대표관리자가 콘솔 접속
2. '멤버' 메뉴 → 본인 이름 옆 '[권한 위임]' 클릭
3. 새 대표관리자 선택
4. 위임 절차 진행

### 주의사항

- 새 대표관리자도 동일 회사 소속이어야 함
- 사업자 인증이 필요할 수 있음
- 약관 재동의 필요

## ❓ 자주 묻는 질문

### Q: 사업자 인증이 반려됐어요

**A**: 반려 사유를 확인하고 다음을 점검하세요:
- 서류가 선명한지 확인
- 정보가 정확히 일치하는지 확인
- 유효 기간이 지나지 않았는지 확인

### Q: 개인 개발자인데 사업자가 필요한가요?

**A**: 수익화가 필요하면 개인 사업자 등록이 필요합니다.
- 국세청 홈택스에서 간편하게 등록 가능
- 무료 앱만 출시한다면 사업자 없이도 가능

### Q: 법인 사업자인데 대표자가 아니에요

**A**: 대리인으로 등록 가능합니다.
- 인감증명서, 위임장, 신분증 사본 필요
- 법인 대표자의 위임을 받아야 함

### Q: 사업자 정보가 변경됐어요

**A**: 콘솔에서 정보 업데이트 후 재검토 요청하세요.
- 상호명, 주소 변경 시 새 서류 필요
- 검토 기간 소요
