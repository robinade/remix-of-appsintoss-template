## 🎨 디자인 가이드라인

### 1. 크기와 여백

```
┌────────────────────────────────┐
│ ┌────────────────────────────┐ │
│ │                            │ │
│ │    여백: 최소 10% 확보     │ │
│ │                            │ │
│ │    ┌──────────────┐       │ │
│ │    │              │       │ │
│ │    │    로고      │       │ │
│ │    │              │       │ │
│ │    └──────────────┘       │ │
│ │                            │ │
│ └────────────────────────────┘ │
└────────────────────────────────┘
        600 x 600px
```

- 로고가 꽉 차지 않도록 여백 확보
- 가장자리에 너무 붙지 않게 배치

### 2. 배경색 선택

**권장 배경색:**
- 브랜드 메인 컬러
- 로고와 대비되는 색상
- 라이트/다크 모드 모두 잘 보이는 색상

**피해야 할 배경색:**
- 순수 흰색 (#FFFFFF) - 라이트 모드에서 구분 어려움
- 순수 검정 (#000000) - 다크 모드에서 구분 어려움
- 너무 연한 색상

### 3. 로고 배치

```
┌────────────────────────────────┐
│                                │
│         권장: 중앙 배치         │
│                                │
│           ┌──────┐            │
│           │ LOGO │            │
│           └──────┘            │
│                                │
│                                │
└────────────────────────────────┘
```

- **중앙 배치** 권장
- 한쪽으로 치우치면 앱에서 어색해 보임

### 4. 기존 로고가 있는 경우

로고에 이미 배경이 포함된 경우:
1. 600x600px에 맞게 리사이징
2. 배경까지 포함하여 꽉 차게 배치
3. 추가 여백 불필요

예: 애니팡 스타일 (로고 자체에 배경 포함)

## 📤 아이콘 적용 방법

### 1. CDN에 업로드

아이콘을 CDN에 업로드합니다:
- AWS S3
- Cloudflare R2
- Vercel Blob
- GitHub (public repo)

### 2. granite.config.ts에 설정

```typescript
// granite.config.ts
brand: {
  icon: 'https://your-cdn.com/app-icon-600x600.png',
}
```

### 3. 앱인토스 콘솔에 업로드

1. 콘솔 접속
2. 앱 정보 → 앱 로고
3. 동일한 이미지 업로드

## ❓ 자주 묻는 질문

### Q: 둥근 모서리 로고를 사용해도 되나요?

**A**: 안 됩니다. 앱인토스에서 자동으로 라운드 처리하므로, 원본은 각진 형태여야 합니다.

### Q: 로고가 정사각형이 아니에요

**A**: 배경을 추가하여 600x600px 정사각형으로 만드세요.
1. 배경색 선택
2. 로고를 중앙에 배치
3. 적절한 여백 확보

### Q: 아이콘이 검토에서 반려됐어요

**A**: 다음을 확인하세요:
- 규격이 정확한지 (600x600px, PNG)
- 투명 배경이 아닌지
- 모서리가 각진 형태인지
- 다른 앱과 혼동되지 않는지
