# 📋 앱인토스 콘솔 등록 가이드

> 앱인토스 콘솔에서 앱을 등록하고 관리하는 방법을 안내합니다.

---

## 🔗 콘솔 접속

**앱인토스 콘솔**: https://console-apps-in-toss.toss.im

---

## 1️⃣ 회원가입

### 가입 조건
- 만 **19세 이상**
- **본인 명의** 토스 앱 필요

### 가입 방법
1. 콘솔 접속
2. '회원가입' 클릭
3. 토스 앱으로 본인 인증
4. 토스 비즈니스 회원 가입 완료

---

## 2️⃣ 워크스페이스 생성

### 워크스페이스란?
팀원들이 함께 앱을 관리할 수 있는 공간입니다.

### 생성 방법
1. 로그인 후 '워크스페이스 만들기' 클릭
2. 워크스페이스 이름 입력 (예: "우리팀 프로젝트")
3. 생성 완료

### 주의사항
- 사업자당 **1개의 워크스페이스**만 사용 가능
- 이름은 나중에 수정 가능
- 중복 이름 불가

---

## 3️⃣ 멤버 초대 (선택)

### 권한 종류

| 권한 | 설명 |
|------|------|
| **대표관리자** | 모든 권한 (자동 부여) |
| **관리자** | 멤버 관리, 설정 변경 가능 |
| **구성원** | 공동 작업만 가능 |

### 초대 방법
1. '멤버' 메뉴 클릭
2. '+초대하기' 클릭
3. 이메일 입력 + 권한 선택
4. 초대 완료

---

## 4️⃣ 앱 등록

### 등록 시작
1. '앱' 메뉴 클릭
2. '+등록하기' 클릭

### 기본 정보 입력

#### ① 앱 로고
| 항목 | 규격 |
|------|------|
| 크기 | 600x600px |
| 형식 | PNG (투명 배경 불가) |
| 모서리 | 각진 형태만 가능 |

> 📥 [아이콘 가이드 다운로드](https://developers-apps-in-toss.toss.im/resources/prepare/logo_guide.pdf)

#### ② 앱 이름 (한글)
- 토스 앱에 표시되는 이름
- 서비스 특성이 바로 이해되도록 작성
- 예: "할일 관리", "쇼핑몰", "퍼즐 게임"

#### ③ appName (영문)
```
⚠️ 매우 중요!
```
- granite.config.ts의 appName과 **정확히 일치** 필수
- 영문 소문자, 숫자, 하이픈만 사용
- 한 번 등록하면 **수정 불가**
- 딥링크 주소로 사용: `intoss://{appName}`

#### ④ 사용 연령
- 현재 만 19세 이상만 지원

#### ⑤ 고객센터 정보
- 이메일 (필수)
- 연락처 (선택)
- 채팅 상담 주소 (선택)

---

## 5️⃣ 앱 정보 검토 요청

### 검토 요청
1. 모든 정보 입력 완료
2. '검토 요청하기' 버튼 클릭

### 검토 기간
- 영업일 기준 **1~2일** 소요
- 결과는 콘솔 + 이메일로 안내

### 검토 통과 후
- 사업자 등록 진행 (수익화 필요 시)
- 앱 번들 업로드 가능

---

## 6️⃣ 앱 출시 요청

### 앱 번들 업로드
1. 로컬에서 빌드: `npm run granite:build`
2. 생성된 `.ait` 파일 확인
3. 콘솔에서 '앱 번들' 메뉴 → '업로드'
4. 버전 입력 (예: 1.0.0)

### 출시 검토 요청
1. '출시' 메뉴 클릭
2. 업로드한 버전 선택
3. '검토 요청하기' 클릭

### 출시 검토 기간
- 영업일 기준 **1~2일** 소요
- 반려 시 사유 안내 (콘솔 + 이메일)

### 출시 완료
- 검토 통과 시 자동 출시
- 토스 앱에서 앱 접속 가능

---

## 🔄 앱 업데이트

### 업데이트 방법
1. 코드 수정 후 빌드: `npm run granite:build`
2. 콘솔에서 새 버전 업로드
3. 출시 검토 요청
4. 검토 통과 후 자동 배포

### 롤백
- 콘솔에서 이전 버전으로 롤백 가능
- 긴급 상황 시 빠르게 복구

---

## ❓ 자주 묻는 질문

### Q: appName을 잘못 입력했어요
A: appName은 수정 불가합니다. 새로운 앱을 등록해주세요.

### Q: 앱 정보 검토가 반려됐어요
A: 반려 사유를 확인하고 수정 후 재요청하세요.

### Q: 워크스페이스를 삭제하고 싶어요
A: 콘솔 고객센터로 문의해주세요.
