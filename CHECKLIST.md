## 🔴 필수 항목 (반드시 완료해야 함)

### 앱 정보

- [ ] **앱 아이콘 준비** (600x600px, PNG, 정방형, 모서리 둥글게 불가)
- [ ] **granite.config.ts의 `icon` 설정** - 아이콘 URL 입력
- [ ] **granite.config.ts의 `appName` 설정** - 콘솔 등록 이름과 일치
- [ ] **granite.config.ts의 `displayName` 설정** - 앱 이름 입력
- [ ] **granite.config.ts의 `primaryColor` 설정** - 브랜드 컬러 입력
- [ ] **src/template.config.ts 동기화** - granite.config.ts와 일치

### 콘솔 등록

- [ ] **앱인토스 콘솔 가입** 완료
- [ ] **워크스페이스 생성** 완료
- [ ] **앱 등록** 완료 (appName 일치 확인!)
- [ ] **앱 정보 검토** 통과

### 기술 요구사항

- [ ] **라이트 모드만 지원** (다크 모드 CSS 없음)
- [ ] **핀치줌 비활성화** (index.html의 viewport 설정 확인)
- [ ] **커스텀 네비게이션 바 없음** (앱인토스가 자동 제공)
- [ ] **외부 링크/자사 앱 유도 없음**
- [ ] **터치 영역 44px 이상** 확보

### 테스트

- [ ] **샌드박스 테스트** 완료
- [ ] **QR 테스트** 1회 이상 완료 (출시 검토 요청 조건)

### 빌드

- [ ] **`npm run granite:build` 성공**
- [ ] **번들 용량 100MB 이하** (압축 해제 기준)

- [ ] **사업자 정보 등록** 완료
- [ ] **사업자 인증** 완료

- [ ] **약관 동의** 완료

### 토스 로그인 사용 시

- [ ] **콘솔에서 토스 로그인 설정** 완료
- [ ] **약관 등록 및 연결** 확인
- [ ] **로그인 인트로 화면** 제공 (서비스 소개 필수)

### 토스페이 결제 사용 시

- [ ] **콘솔에서 토스페이 설정** 완료
- [ ] **주문 금액 = 결제창 금액** 일치 확인
- [ ] **결제/취소 정상 동작** 확인
- [ ] **결제 실패 오류 메시지** 안내
- [ ] **결제 내역 조회** 가능

### 인앱 결제 사용 시

- [ ] **콘솔에서 인앱 결제 설정** 완료
- [ ] **상품 금액 = 스토어 결제창 금액** 일치 확인
- [ ] **결제/취소 정상 동작** 확인
- [ ] **기기 변경 후 데이터 유지** 확인
- [ ] **구독 상품 미사용** (현재 지원 안 함)

### 인앱 광고 사용 시

- [ ] **콘솔에서 인앱 광고 설정** 완료
- [ ] **광고 사전 로드 (Preload)** 구현
- [ ] **리워드 광고: 완료 후에만 보상 지급**
- [ ] **광고 종료 이벤트 처리** 구현
- [ ] **광고 종료 후 미니앱 정상 복귀**
- [ ] **테스트 광고 ID 제거** (프로덕션 ID 사용)

### 푸시/알림 사용 시

- [ ] **콘솔에서 푸시 설정** 완료
- [ ] **기능성 메시지만 발송** (필수 알림만)
- [ ] **푸시 템플릿 정상 동작** 확인

### 게임 앱인 경우

- [ ] **게임 등급 분류** 완료 (게임물관리위원회 또는 스토어)
- [ ] **등급 정보 콘솔에 등록**
- [ ] **리더보드 설정** (사용 시)

## 📋 자동 검증 명령어

```bash
# 출시 준비 상태 자동 검증
npm run check:publish
```

이 명령어로 다음 항목을 자동으로 확인합니다:
- granite.config.ts 필수 항목 설정 여부
- template.config.ts 동기화 상태
- 빌드 성공 여부
- 용량 제한 준수 여부
